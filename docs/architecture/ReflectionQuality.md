# Reflection Quality V1

## 목적

Today's Reflection는
사용자를 평가하거나 정의하는 문장이 아니다.

여러 분석 결과를 하나의 자연스러운 이야기로
연결하여 사용자가 스스로 자신의 흐름을
돌아볼 수 있도록 돕는 것이 목적이다.

# 금지사항

Reflection는

사용자를 정의하지 않는다.

사용자를 평가하지 않는다.

행동을 지시하지 않는다.

동기를 추측하지 않는다.

기록에 없는 사실을 추가하지 않는다.

---
